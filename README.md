# one-hundred-monkeys
How long will it take 100 monkeys to say Roll Tide?

Let's play with golang channels to find out!

First, clone it!

`git clone https://github.com/forrest321/one-hundred-monkeys.git`

Next, get the dependencies

`go mod vendor`

`go mod tidy`

Build it.

`go build`

Run it

`./one-hundred-monkeys`

or

`go run main.go`
